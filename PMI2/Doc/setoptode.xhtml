Define a Grid of Optodes

SetOptode defines a rectangular grid of fibers with constant amplitude. Used mostly for simulations.

Function Summary

Syntax: [Pos, Amp] = SetOptode(X, Y, amp);
Inputs: X Vector of X positions
Y Vector of Y positions
amp Coupling coefficient(s)
Outputs: Pos List of positions
Amp List of amplitudes
