Compute Voxel Locations

Given a Medium.CompVol structure, compute the x y z location of the center of each voxel. Also returns the volume of the voxels.

Function Summary

Syntax: [X,Y,Z,dV] = sampleVolume(Medium.CompVol);
Inputs: Medium.CompVol Medium.CompVol structure
Outputs: X List of X coordinates
Y List of Y coordinates
Z List of Z coordinates
dV The volume of each voxel
